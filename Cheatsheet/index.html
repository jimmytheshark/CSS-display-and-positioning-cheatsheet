CSS Display and Positioning

Display Values

Name Tags Attributes
:block block Block-level elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted.
:inline inline Inline elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted.
:inline-block inline-block Inline-block elements can appear next to each other, and can have their width and height manually adjusted.

Positioning Values

Name Tags Attributes
:static static Default positioning value
:relative relative enables an element to be positioned relative to where it would have originally been on a web page.
:absolute absolute enables an element to ignore sibling elements and instead be positioned relative to its closest parent element that is positioned with relative or absolute
:fixed fixed it is set/pinned to a specific spot on a page.

Other Properties

Name Tags Attributes
z-index: z-Index A property that specifies how far back or how far forward an element will appear on a web page when it overlaps other elements.
clear: clear The CSS clear property specifies how an element should behave when it bumps into another element within the same containing element.
float: float The CSS float property determines how far left or how far right an element should float within its parent element.
